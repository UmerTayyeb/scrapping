Package scrape handles scraping-related tasks. It first scrapes articles from the main page of the main URL, then scrapes data from each article and calls Insert_article_metadata to store the data in the database.

func Scrape_articles()

Scrape_articles scrapes articles from a specified URL and inserts metadata into the database. URL and allowed domains are loaded from environment variables (TARGET_URL, ALLOWED_DOMAINS).
